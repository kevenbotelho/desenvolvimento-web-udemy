Posicionamentos ( static, relative, absolute e fixed )

Static = padrão - aparece na ordem que foi definida no código | um abaixo do outro

Relative = Acompanha as seguintes propriedades: top;bottom;right;left.
Ela por padrão é relativa ao body, ou seja, toma ele como referencia pra se deslocar

Absolute = Sai do fluxo padrão do site, vira uma camada sobreposta ao site. Tambṕem podemos usar as propriedades: top;bottom;right;left.
Posição calculada com base no conteiner que está o elemento, normalmente é o body
